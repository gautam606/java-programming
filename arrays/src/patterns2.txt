What Problem Does It Solve?
Kadane’s Algorithm finds the maximum sum of a subarray (a continuous part of the array).
This is also called the Maximum Subarray Problem.

✅ Easy Example
Code
arr = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
We want the subarray with the largest sum.

🧠 Kadane’s Idea (in plain words)
Walk through the array one element at a time.

Keep track of:

currentSum → best sum ending at this position.

maxSum → best sum found overall.

If currentSum becomes negative, reset it to 0 (because starting fresh is better).

Update maxSum whenever you find a bigger sum.

✅ Step-by-Step Dry Run
Start: currentSum = 0, maxSum = -∞.

Take -2: currentSum = max(0 + (-2), -2) = -2 → reset to 0.

Take 1: currentSum = 1 → maxSum = 1.

Take -3: currentSum = -2 → reset to 0.

Take 4: currentSum = 4 → maxSum = 4.

Take -1: currentSum = 3 → maxSum = 4.

Take 2: currentSum = 5 → maxSum = 5.

Take 1: currentSum = 6 → maxSum = 6.

Take -5: currentSum = 1 → maxSum = 6.

Take 4: currentSum = 5 → maxSum = 6.

👉 Final Answer: maxSum = 6
Subarray = [4, -1, 2, 1].
